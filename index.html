Rock API Documentation

This API allows you to manage bands and their albums, providing functionality for creating, retrieving, updating, and deleting records.

Get All Bands

http://localhost:5000/api/bands

Retrieves a list of all bands in the database. Each band is represented by its ID, name, and the year it was formed.

Get Band by ID

http://localhost:5000/api/bands/:bandID

Fetches detailed information about a specific band by its ID. bandID is the id of the band you want. If the band your trying to find isnt there it will give you an error message.

Get All Albums

http://localhost:5000/api/albums

Retrieves a list of all albums in the database. Each album is represented by its ID, title, associated band ID, and release year.

Get Album by ID

http://localhost:5000/api/albums/:albumID

Fetches detailed information about a specific album by its ID. albumID is the id of the album you want. If the album your trying to find isnt there it will give you an error message

ADMIN

Add a New Band

http://localhost:5000/api/bands/add/:name/:formed/:pass

Creates a new band entry in the database. This requires authentication via a password.

Add a New Album

http://localhost:5000/api/albums/add/:title/:band_id/:released/:pass

Creates a new album entry in the database, associating it with a specific band.

Delete a Band

http://localhost:5000/api/bands/:id/delete/:pass

Deletes a band from the database using its ID. This action requires authentication via a password.

Delete an Album

http://localhost:5000/api/albums/:id/delete/:pass

Deletes an album from the database using its ID. This action requires authentication via a password.

Update a Band

http://localhost:5000/api/bands/:id/update/:name/:formed/:pass

Updates an existing band’s details using its ID.

Update an Album

http://localhost:5000/api/albums/:id/update/:title/:band_id/:released/:pass

Updates an existing album’s details using its ID.
